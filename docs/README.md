# API docs

- **For the Cult API.md** — Long-form API documentation.
- **openapi.json** — OpenAPI 3.0 spec (generated by webapp: `bun run generate:openapi` from the webapp directory).
- **index.html** — Redoc viewer; open locally to browse the spec (uses `openapi.json`).

The live store serves the spec at `GET /api/openapi.json` and the webapp’s API docs UI at `/api/docs`.
