# For the Cult — Overview

An ecommerce store that sells apparel, accessories, and lifestyle products. Accepts both card and cryptocurrency payments.

## For AI agents

- This site exposes a REST API designed for AI agents: natural-language product search, structured errors with next-step suggestions, action hints in responses, and multiple payment methods.
- **Start here:** `GET /api/agent/capabilities` — returns supported features, payment options, shipping countries, rate limits, and a complete endpoint reference.
- **OpenAPI spec:** `GET /api/openapi.json`
- **Interactive docs:** `/api/docs` (Swagger UI)
- **LLM API summary:** `/llm/api.txt`

## Key concepts

- **Products** — Published products with variants (size, color), categories, and brands. Search via `POST /api/products/semantic-search` (natural language) or `POST /api/products/search` (structured filters).
- **Checkout** — `POST /api/checkout` with items, email, payment method, and shipping address. Returns an order ID and payment instructions.
- **Orders** — Poll `GET /api/orders/{orderId}/status` until the order is confirmed. Fulfillment partners handle shipping.
- **Payments** — Accepts card (Stripe) and cryptocurrency (Solana, Ethereum, Base, Arbitrum, Polygon, Bitcoin via BTCPay).

## Quick start (3 calls)

1. **Find products:** `POST /api/products/semantic-search` with `{ "query": "lightweight running shoes" }`
2. **Create order:** `POST /api/checkout` with items, email, payment method, and shipping address
3. **Confirm payment:** Poll `GET /api/orders/{orderId}/status` every 5 seconds until status is `paid`

## Customer-facing

- Storefront with product pages, cart, checkout, and order tracking.
- Account sign-in via email, Google, GitHub, or wallet.
- Dashboard for order history, notifications, and support.

For the full API reference, see `/llm/api.txt` or call `GET /api/agent/capabilities`.
