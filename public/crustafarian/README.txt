Design files for Crustafarian Printify products. The design file must match the product type.

Naming: use lowercase with hyphens, e.g.:
  sticker.png
  mug.png
  poster.png
  phone-case.png
  shot-glass.png
  playing-cards.png
  throw-blanket.png
  water-bottle.png
  etc.

If a product-specific file is missing, the script falls back to public/crustafarian.png.

Supported extensions: .png, .jpg, .webp
